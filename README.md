# Vanilla JS To-Do App 📝

A lightweight, interactive To-Do list application built to solidify foundational web development concepts. 

🚀 Features
Add Tasks: Quickly add new items to your daily list.
Delete Tasks: Remove completed or unwanted tasks individually.
Dynamic UI: Interface updates instantly without page reloads.

🛠️ Tech Stack
HTML5: Semantic structure.
CSS3: Styling and layout.
JavaScript (Vanilla): Core logic and interactivity.

💡 What I Learned
Building this project helped reinforce my understanding of core JavaScript concepts, specifically:
DOM Manipulation: Selecting elements, creating new nodes (`document.createElement`), and updating the interface dynamically.
Event Delegation: Utilizing event bubbling by attaching a single event listener to the parent `<ul>` element to efficiently handle clicks on dynamically generated "Delete" buttons.
